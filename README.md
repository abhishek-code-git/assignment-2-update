Project: Dubai Mall Interactive Sales Deck

Overview:
This project is a cinematic, interactive sales tool designed to showcase the Dubai Mall as a global retail and entertainment destination. It replaces traditional pitch decks with a dynamic, video-first experience.

Tech Stack:

Next.js
Tailwind CSS
Framer Motion

Features:

Video-first storytelling
Non-linear navigation
Interactive sections
Event booking module

AI Usage:

Generated visuals using AI tools
Assisted content generation

Deployment:
Live on Vercel
